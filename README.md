## Usage

`go run main.go 1.png 2.png 3.png`

## Build

`go build -o convert_to_jpegli main.go`

## Add to zsh

`nvim ~/.zshrc`

`alias convert_to_jpegli="$HOME/ <add paths> /convert_to_jpegli"`
